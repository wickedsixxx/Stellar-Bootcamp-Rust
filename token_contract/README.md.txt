# Token Contract

A simple token contract for the Stellar Soroban smart contract platform.

## Features
- `initialize`: Set admin for the token.
- `mint`: Admin can mint new tokens to an address.
- `transfer`: Transfer tokens between users.
- `balance`: Query balance of an address.

## Tech
- Written in Rust
- Built with Soroban SDK v20
- Supports persistent storage

## Folder Structure
- `src/lib.rs`: Smart contract code
- `src/test.rs`: Contract test cases
